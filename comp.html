Complex Operations

Complex Operations are operations that the CPU cannot execute on it's own. This includes operations that require multiple logic circuits. The way computers complete these operations is by: Loading a Value (from Memory), Executeing a simple Operation (NOT, AND, OR, etc- just one), Load result to Memory- Then repeat. This allows for operations such as Subtraction, Multiplication, Exponents, Line drawing, etc- which require recursive and repetitive calculations.

Examples:

  • Multiplication: Computers cannot multiply- but they can add. So, the ALU will add 'y' to 'y' 'x' number of times. For an explanation (that is far better than mine) check this out
  • Subtraction: Computers cannot subtract either- but they can 'comprehend' negative numbers. A negative number can be found by inverting a number (call it y), then adding one. This causes an overflow (a number 1 bit or more larger than the computer can comprehend) but in this case we can ignore it. Then, adding the result (-y) to x, we get x-y.
